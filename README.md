# School Management System

A web-based School Management System developed using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.

## Features
- Student Management
- Attendance Management
- Marks Management
- Result Management
- Class Timetable Management
- Exam Timetable Management
- Admin & Student Login
- Role-Based Access Control

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript
